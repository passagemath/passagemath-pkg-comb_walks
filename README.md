
# **Walk Models with elliptic kernel function** 

`daeg1` is a Sage package for understanding and getting data for generating functions of Walks in the Quarter plane where the kernel function describes an elliptic curve.

## **1. Installing the package**
Once the repository has been cloned on the folder `###`, run the Sage command

  `sage -pip install ###/sage`

and after the installation the user is ready to load the package from Sage typing the following command:

  `from daeg1 import *;`
  

For further information, the user can compile its own version of the documentation running `make doc` on the folder `###/sage` and check there all the documentation
and tests of the code.

## **2. Extra requirements**
There are no extra requirements to running this package. If any dependencies errors are found, please contact the owners for future fixing.
